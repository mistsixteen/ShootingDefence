# ShootingDefence

## Unity3D Shooting-Defence Game Project

### - 3D Top-down Shooter 개발

## 개발 기능
### 추적, 공격, 원거리공격 등 적의 AI 구현, 애니메이션 적용
### 인벤토리/무기 장착 기능 구현
### FSM 패턴을 사용한 인벤토리 관리
### 회복, 아이템획득 등 아이템 오브젝트 구현
### Factory 패턴을 사용한 총알/적 오브젝트 생성
### Object Pooling을 사용한 생성된 리소스 관리
